# 福格行为模型 (Fogg Behavior Model)

> **核心公式**：B = MAP — 行为 = 动机 × 能力 × 提示
> 三个条件同时满足，行为才会发生。缺任何一个，行为都不会发生。

---

## 模型概述

```
行为（Behavior）= 动机（Motivation）× 能力（Ability）× 提示（Prompt）

B = M × A × P

- 动机（M）：你想做这件事的程度
- 能力（A）：做这件事的难度（越简单越好）
- 提示（P）：触发你去做的信号
```

**关键洞察**：大多数行为改变失败，不是因为缺乏动机，而是因为**能力门槛太高**或**缺少提示**。

---

## 三要素详解

### 1. 动机（Motivation）

动机来源：
- **快感/痛苦**：即时回报 vs 即时惩罚
- **希望/恐惧**：对未来的预期
- **社会认同/社会排斥**：他人的看法

**常见误区**：
- 认为"只要足够想做，就能做到" → 错！动机是波动的
- 依赖动机来维持习惯 → 不可靠，应该降低能力门槛

### 2. 能力（Ability）

影响能力的因素（越少越好）：
- **时间**：需要花多长时间？
- **金钱**：需要花多少钱？
- **体力**：需要多大体力？
- **脑力**：需要多深的思考？
- **社会偏差**：是否违反社会规范？
- **非常规性**：是否打破日常习惯？

**核心策略**：让目标行为**尽可能简单**。

### 3. 提示（Prompt）

三种提示类型：
- **外部提示**：闹钟、日历提醒、他人提醒
- **内部提示**：身体信号（饿了→吃饭）、情绪信号
- **行为锚点**：把新行为绑定到已有习惯上（做完X之后，我就做Y）

## 实践应用

### 想要建立新习惯

```
1. 降低能力门槛（最重要！）
   - "每天跑步1小时" → "每天穿上跑鞋走出门"
   - "每周写一篇博客" → "每天写3句话"
   - "每天读书30分钟" → "每天读1页"

2. 设置明确提示
   - 行为锚点："早上刷完牙后，我就打开笔记写3句话"
   - 外部提示：Vibeflow 番茄钟、日历提醒

3. 动机作为加速器（不是基础）
   - 有动机时多做一点
   - 没动机时做最小版本
   - 永远不要因为"不想做"就完全不做
```

### 想要消除坏习惯

```
1. 增加能力门槛（让坏习惯变难）
   - 刷手机成瘾 → 卸载App / 设置屏幕时间 / 手机放远处
   - 熬夜 → 22:00 设置断网 / 闹钟提醒

2. 移除提示
   - 手机放在视线外
   - 关闭通知
   - 不把零食放在桌上

3. 降低动机
   - 找到替代满足方式
   - 理解坏习惯背后的真实需求
```

### 诊断行为问题

当某个行为做不到时，按顺序检查：

```
Step 1: 有提示吗？
  → 没有 → 设置提示（闹钟/锚点/环境）
  → 有 → 继续

Step 2: 够简单吗？
  → 太难 → 降低门槛（缩小目标到"不可能失败"的程度）
  → 够简单 → 继续

Step 3: 有动机吗？
  → 没有 → 找到与核心目标的关联 / 增加即时回报
  → 有但波动 → 正常！依赖前两步，不要依赖动机
```

## 与 LessNote 的结合

### /morning 中使用

检查今日计划时，用 MAP 验证：
- 这个任务有触发提示吗？（在日历/Vibeflow 里设置了吗？）
- 这个任务的第一步够简单吗？（能在5分钟内开始吗？）
- 如果动机不足怎么办？（最小版本是什么？）

### /focus 中使用

任务启动前的结构化思考可以加入：
- "这个任务的最小可交付版本是什么？"（降低能力门槛）

### 诊断拖延

当某件事反复拖延时，用福格模型诊断：

```markdown
## 拖延诊断

**拖延的事情：** [具体描述]

**MAP 检查：**
- 提示（P）：有没有明确的触发？ [有/无]
  → 如果无：设置什么提示？
- 能力（A）：第一步是什么？需要多久？ [简单/困难]
  → 如果困难：最小版本是什么？
- 动机（M）：这件事和我的核心目标有什么关系？ [强/弱]
  → 如果弱：是否应该放弃这件事？

**行动计划：** [基于诊断的具体下一步]
```

## 核心提醒

1. **不要依赖动机** — 动机是波动的，系统设计比意志力更可靠
2. **让开始变得极其简单** — "穿上跑鞋" 比 "跑5公里" 更容易坚持
3. **庆祝微小胜利** — 完成最小版本后给自己正反馈，强化行为回路
4. **环境设计 > 自我约束** — 改变环境比改变自己更有效

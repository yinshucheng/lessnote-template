## ⚡ 关键原则清单

### 原则一：极端时间压迫感 + 第一性原理

| 场景 | 自问 |
|------|------|
| 制定计划时 | 能否压缩到1/5时间？我在优化流程还是重新定义问题？ |
| 选择项目时 | 这个能带来10倍提升吗？还是只是5-10%的改进？ |
| 遇到困难时 | 我是否在用第一性原理思考？问题的本质是什么？ |
| 学习技术时 | 我能把它"装在脑子里"吗？深度够吗？ |

### 原则二：先做正确的事，再把事情做对

| 场景 | 自问 |
|------|------|
| 开始任务前 | 这是正确的事吗？方向对了吗？ |
| 投入精力时 | 我是否在错误的事情上努力？ |
| 做决策时 | 先问"该不该做"，再问"怎么做好" |
| 复盘总结时 | 我做的是正确的事，还是只是把事情做对了？ |

### 底层原则：责任与个人品牌

> **收入 = 专长 + 责任 + 杠杆**

1. **以个人名义承担风险** - 不躲在公司/团队后面
2. **把自己产品化** - 每个项目都是个人品牌素材
3. **输出即责任** - 不输出就没有信誉，没有信誉就没有回报
4. **失败是代价，不是终点** - 不发才是真正的失败

### 行动决策原则

1. **行动优先** - 信息收集服务于具体行动
2. **目标锚定** - 所有行动都要服务于明确目标
3. **知行合一** - 认知必须转化为行动
4. **真相优先** - 面对不舒服的真相而非舒服的幻觉

### 认知偏误防范

1. **确认偏误** - 主动寻找反对观点
2. **行动偏误** - 区分有效行动和无效忙碌
3. **完美主义** - 用"足够好"替代"完美"
4. **沉没成本** - 基于未来而非过去做决策
